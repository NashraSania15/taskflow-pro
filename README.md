TaskFlow Pro is a task and productivity management mobile application built using Flutter and Firebase.  
The app focuses on clean architecture, scalable code structure, and real-world authentication flows.

## Features
- User authentication (Sign up / Login)
- Task creation and management
- Clean and structured UI
- Firebase integration
- Scalable project architecture

## Tech Stack
- Flutter (Dart)
- Firebase Authentication
- Cloud Firestore
- Material UI

## Project Status
- Version: v1.0
- Platform: Android
- Status: Deployed (Release APK generated)

## Purpose
This project was built to gain hands-on experience in building production-ready mobile applications using Flutter and Firebase, following clean coding practices and real-world workflows.

## Getting Started
1. Clone the repository
2. Run `flutter pub get`
3. Configure Firebase
4. Run the app on an emulator or physical devices
